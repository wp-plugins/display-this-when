This plugin easily allows you to create polls.


== Description ==

Ever wanted to show a particular section of cotent on a certain date but don't want to do it to the whole post or page? "Display This When..." will allow you to block of a section of your content to only show on a set date.

Say if you want enter in an April Fools Day joke into your post, all you need do is this:

[displaythis when="1-4"]Chinchillas know how to use dynamite![/displaythis]

And thus chinchillas will use dynamite on the 1st of April.


== Installation ==

1. Download and unzip
1. Upload to your wp-content/plugins folder
1. Activate from the dashboard
1. Go to the new Polls menu page
